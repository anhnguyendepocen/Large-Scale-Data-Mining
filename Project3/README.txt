Each section of the project is done in a separate iPython file. The iPython files belonging to this project are as follows:
1) Part1.pynb : Questions 1-6
2) KNN_Prediction&Ranking.ipynb : For KNN Collaborative filter
3) nnmf.ipynb : For NNMF Collaborative filter
4) MF-with-bias-part.ipynb : For MF with bias Collaborative filter
5) NCFilter.ipynb : For Naive Collaborative filter

The question numbers have been written as a comment in each cell of the respective ipynb files.
