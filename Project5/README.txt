Each section of the project is done in a separate iPython file inside "Code" folder. The folder has a file "generating_csv.ipynb" which contains the code for generating a CSV file corresponding to each hashtag. This has been done to speed up the loading of data. The output of running this file for each hashtag needs to be included in a "csv" folder which must be present in the same directory as other task files.

The iPython files belonging to this project are as follows:
1. Task1.1.ipynb

4. Task1.4.ipynb
